# Cookie Recipe Page

A simple, responsive recipe page built with HTML and CSS, showcasing a Classic Chocolate Chip Cookies recipe.

## Overview

This project displays a recipe card layout including a photo, description, preparation time, ingredients list, step-by-step instructions, and a nutrition breakdown table.

## Features

- Clean, card-style layout with rounded corners and soft shadows
- Responsive image with adjustable size using `object-fit`
- Highlighted preparation time box
- Organized ingredients and instructions sections
- Nutrition facts table
- Beginner-friendly CSS with no advanced frameworks

## Technologies Used

- HTML5
- CSS3 (plain CSS, no frameworks)

## File Structure

```
cookie-recipe.html
```

All HTML and CSS are contained in a single file for simplicity.

## How to View

1. Download or clone the project files.
2. Open `cookie-recipe.html` in any web browser.

## Sections Breakdown

- **Header Image**: Displays the cookie photo
- **Title & Description**: Name and short intro of the recipe
- **Preparation Time**: Total, prep, and baking time
- **Ingredients**: List of required ingredients
- **Instructions**: Step-by-step cookie-making process
- **Nutrition**: Calorie and macro breakdown per cookie
